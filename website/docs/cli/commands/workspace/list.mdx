# Command: workspace list

The `mnptu workspace list` command is used to list all existing workspaces.

## Usage

Usage: `mnptu workspace list [DIR]`

The command will list all existing workspaces. The current workspace is
indicated using an asterisk (`*`) marker.

## Example

```
$ mnptu workspace list
  default
* development
  jsmith-test
```
