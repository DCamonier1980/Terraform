# Command: workspace

The `mnptu workspace` command is used to manage
[workspaces](/mnptu/language/state/workspaces).

This command is a container for further subcommands that each have their own page in the documentation.

## Usage

Usage: `mnptu workspace <subcommand> [options] [args]`

Choose a subcommand page for more information.
