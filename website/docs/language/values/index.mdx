# Variables and Outputs

The mnptu language includes a few kinds of blocks for requesting or
publishing named values.

- [Input Variables](/mnptu/language/values/variables) serve as parameters for
  a mnptu module, so users can customize behavior without editing the source.

- [Output Values](/mnptu/language/values/outputs) are like return values for a
  mnptu module.

- [Local Values](/mnptu/language/values/locals) are a convenience feature for
  assigning a short name to an expression.
